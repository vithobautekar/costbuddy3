To point this app at a different Firebase project

Edit the firebaseConfig object near the top of the <script> tag in index.html with your project's config (Firebase Console → Project settings → General → Your apps → SDK setup and configuration), then re-deploy the file.
Firestore security rules

Paste the rules from firestore.rules (bundled alongside this file, and repeated in the comment block at the bottom of index.html) into Firebase Console → Firestore Database → Rules.
